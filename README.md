# 自動化檢測網站內容

**功能 : 檢查網頁元素是否存在，不論成功或失敗都產生HTML格式的報告內容**

&emsp;
&emsp;

## 開發環境
程式語言 : Python 3.8.8

使用套件 : selenium 4.15.0 + xtestrunner 1.7.0
