# 自動化檢測網站內容

**功能 : 檢查網頁元素是否存在，不論成功或失敗都產生截圖的HTML格式報告**

**撰寫 : 吳彥楓**

&emsp;
&emsp;

## 開發緣由
學習 selenium 作為取代人工檢閱網站的方式，大致完成了自動化作業，包含測試 / 產出報告，寄送報告郵件。 

## 開發環境
程式語言 : Python 3.8.8

使用套件 : selenium 4.15.0 + xtestrunner 1.7.0
